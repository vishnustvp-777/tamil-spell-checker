# Local Tamil Spell Checker

This is a simple, locally-run Tamil language spell checker built with Python. This project checks if a Tamil word exists in a local dictionary and suggests close matches if the word is misspelled.

## Features
* Loads a custom, extensible Tamil dictionary.
* Uses character similarity to suggest corrections for misspelled words.
* Runs entirely offline.
